# مكتبة أوبلس الرقمية

موقع ثابت (Static Site) لتوثيق كل الملفات اللي بنعملها لصيدليات أوبلس — تقارير، لوحات، دراسات، أدوات — في مكان واحد بيتحدث أول بأول.

## إزاي تضيف ملف جديد

1. حط الملف نفسه (PDF / XLSX / PPTX / HTML) جوه مجلد `files/`
2. افتح `index.html` ودوّر على مصفوفة `const files = [...]` في آخر الكود
3. ضيف سطر جديد بنفس الشكل ده:

```js
{
  title: "اسم الملف",
  description: "وصف سطر واحد",
  category: "استراتيجية",   // أو: تقارير | دراسات | أدوات (أو أي تصنيف جديد)
  type: "PDF",               // أو: XLSX | PPTX | HTML
  date: "2026-07-21",
  href: "files/اسم-الملف.pdf"
},
```

4. احفظ وارفع (commit + push) على GitHub — Render هيعمل Deploy تلقائي خلال دقيقة أو اتنين.

مفيش حاجة تانية مطلوبة — التصنيفات والبحث بيشتغلوا تلقائي من نفس المصفوفة دي.
