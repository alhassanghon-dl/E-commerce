# مكتبة أوبلس الرقمية

موقع ثابت (Static Site) لتوثيق كل الملفات اللي بنعملها لصيدليات أوبلس — تقارير، لوحات، دراسات، أدوات — في مكان واحد بيتحدث أول بأول.

## إزاي تضيف ملف جديد

1. ارفع الملف نفسه (PDF / XLSX) في **نفس مكان** `index.html` مباشرة (مفيش مجلد فرعي مطلوب)
2. افتح `index.html` ودوّر على مصفوفة `const files = [...]` في أول كود الجافاسكريبت
3. ضيف كتلة جديدة بنفس الشكل ده:

```js
{
  title: "اسم الملف",
  description: "وصف سطر أو اتنين",
  category: "استراتيجية",   // أو: تقارير | دراسات | أدوات | فواتير | عروض أسعار (أو أي تصنيف جديد)
  type: "PDF",               // أو: XLSX
  date: "2026-07-21",
  href: "اسم-الملف.pdf"
},
```

4. احفظ وارفع (commit + push) على GitHub — Render هيعمل Deploy تلقائي خلال دقيقة أو اتنين.

مفيش حاجة تانية مطلوبة — التصنيفات والبحث بيشتغلوا تلقائي من نفس المصفوفة دي.

**الأسهل:** ابعت الملف هنا في الشات وأنا أظبط كل حاجة وأرجعلك `index.html` جاهز + الملف نفسه.

## طريقة فتح الملفات

الملفات بتتفتح **جوه الموقع نفسه** (مش تنزيل ولا تبويب جديد):
- ملفات PDF بتتعرض مباشرة في نفس الصفحة.
- ملفات XLSX بتتحول لجدول HTML داخل الصفحة (عن طريق مكتبة SheetJS، بتحتاج اتصال إنترنت عادي بس مش أي إعداد إضافي).

## رمز الدخول

الموقع محمي برمز دخول بسيط (`180790` حاليًا). لو عايز تغيّره، دوّر على `GATE_PASSWORD` في أول كود الجافاسكريبت وغيّر القيمة.

ملحوظة: ده حماية بسيطة بس (بتمنع الزيارة العشوائية بالرابط)، مش تشفير حقيقي — أي حد عنده خبرة تقنية بسيطة يقدر يتخطاه من كود الصفحة. لو محتاج حماية أقوى فعليًا، فيه خيار تاني (PageCrypt) يحتاج خطوة إعداد إضافية على Render.
